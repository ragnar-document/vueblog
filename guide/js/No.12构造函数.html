# NO.13 了解构造函数

# 简介

简单的来说就是创建一个共有常用函数，因为多次出现需要多次创建类似的对象所以我们就可以使用构造函数和'new'操作符来实现

# 约定

  1. 构造函数函数名首字母大写
  2. 必须使用new来执行

# 实例

function Into(name,age){
	this.name = name;
  this.age = age;
}

let test = new Into('rangar',20)
let test1 = new Into('baby',1)
