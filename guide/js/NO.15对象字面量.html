# NO.15 变量声明语块和对象字面量

var sayHello = 'hello word'
1

Javascript 使用关键字var来声明变量，变量可以包含任意类型的数据，数组，整数，浮点数，字符串等，没有指定变量的类型，因此JavaScript是一种宽松类型的语言，即便是变量赋了值也可以通过赋予不同的类型值而改变所以JavaScript也算是一门动态语言可以使用var同时声明多个JavaScript变量并赋值

var Person = {
	name:'ragnar',
  age:20,
  sex:'男'
}
1

3
4

对象字面量是指用大括号括起来的一组以逗号分隔的属性所定义的对象属性用冒号设置值而不是等号。对象字面量也可以包括数组，他使用了方括号括起来的一组以逗号分隔的成员可以将属性值设置为函数来定义方法
