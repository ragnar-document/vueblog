# Element UI库

介绍

Element，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库

安装

npm i element-ui -S
1

插件：如果使用VUEcl3的话

vue create my-app
cd my-app
vue add element
1
2
3

在 main.js 中写入以下内容：通过插件安装就无需

import Vue from 'vue';
import ElementUI from 'element-ui';
import 'element-ui/lib/theme-chalk/index.css';
import App from './App.vue';

Vue.use(ElementUI);

new Vue({
  el: '#app',
  render: h => h(App)
});
