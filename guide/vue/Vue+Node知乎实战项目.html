# 知乎API

V3**:https://www.zhihu.com/api/v3

首页热榜：/feed/topstory/hot-lists/total?limit=50&desktop=true

子分类渲染：/feed/topstory/hot-lists?desktop=true

热榜下的子分类：/feed/topstory/hot-lists/${name}?limit=50&desktop=true

首页推荐：/feed/topstory/recommend?limit=10&desktop=true

**V4**:https://www.zhihu.com/api/v4/favlists/discover

相关话题：/questions/275409253/similar-questions

相关推荐：/market/rhea/questions/questions_id/qa_related?answer_id=id

被收藏：/answers/${questions_id}/favlists?include=data%5B*%5D.answer_count%2Cfollower_count%2Ccreator&offset=0&limit=10

热门搜索：/search/top_search

专题页面：/news_specials/list?limit=10&offset=10

最新专题：/news_specials/list

热门收藏列表：/favlists/discover

表情包：/stickers

**其他**

热门收藏未知：https://www.zhihu.com/api/v4/favlists/discover?include=data%5B%2A%5D.total_count%2Ccreator%2Cfollower_count%2Cis_following%3Bdata%5B%2A%5D.favitems%5B%2A%5D.content.question%2Cexcerpt%2Cvoteup_count%2Ccomment_count&limit=10&offset=10

专栏内容：https://zhuanlan.zhihu.com/api/columns/${name}

知乎专栏卡片：https://zhuanlan.zhihu.com/api/recommendations/columns?limit=8&offset=8&seed=7

专栏详情：https://zhuanlan.zhihu.com/api/columns/mandy

专栏下的文章：https://zhuanlan.zhihu.com/api/columns/mandy/articles
