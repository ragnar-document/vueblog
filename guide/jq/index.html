# Jquery库框架

创始人是美国John Resig所创建的一个JavaScript库里面封装了JavaScript的一些方法方便我们使用提高了编码效率。当然时代在变化许许多多库如雨后春笋般冒出。作为老牌JavaScript库的jquery在地位上也面临了危险但是依旧还是有许多公司是在使用jquery的如果说担心淘汰的话是完全没有必要的，因为他的确很好用

jQuery 是一个高效、精简并且功能丰富的 JavaScript 工具库。它提供的 API 易于使用且兼容众多浏览器，这让诸如 HTML 文档遍历和操作、事件处理、动画和 Ajax 操作更加简单。

CDN: https://cdn.bootcss.com/jquery/3.4.1/jquery.js

# Bootstrap

Bootstrap 是全球最受欢迎的前端组件库，用于开发响应式布局、移动设备优先的 WEB 项目。

Bootstrap 是一套用于 HTML、CSS 和 JS 开发的开源工具集。利用我们提供的 Sass 变量和大量 mixin、响应式栅格系统、可扩展的预制组件、基于 jQuery 的强大的插件系统，能够快速为你的想法开发出原型或者构建整个 app 。
